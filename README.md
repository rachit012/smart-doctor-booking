# Smart Doctor Booking 

## Authentication
- Patient registration and login
- Doctor registration and login
- JWT-based secure authentication
- Role-based navigation

## Patient Features
- View nearby doctors using GPS
- Sort doctors by distance and consultation fee
- Search by doctor name or speciality
- View doctor profile and availability
- Book appointment
- Cancel appointment
- Receive booking and reminder notifications
- View appointment history

## Doctor Features
- Add availability slots manually
- View upcoming and past appointments
- Appointment auto-cancel handling
- Slot reopen after cancellation

## Appointment Management
- Real-time slot locking
- Appointment status handling (Pending, Confirmed, Cancelled, Completed)
- Patient-initiated cancellation
- Doctor-side appointment visibility

## Notifications
- Appointment confirmation notification
- Google Maps redirection
- Scheduled reminder (1 hour before appointment)

## Location & Distance
- GPS-based nearby doctor discovery
- Distance calculation from patient to doctor
- Sorting by nearest doctor

## Tech Stack
- Android (Kotlin, Jetpack Compose)
- MVVM Architecture
- Dagger Hilt
- Retrofit + OkHttp
- Node.js Backend
- MongoDB
